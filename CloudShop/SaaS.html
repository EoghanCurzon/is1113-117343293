Software as a Service



What is Software as a Service?

Software as a service (or SaaS) is a way of delivering centrally hosted applications over the Internet as a service. SaaS applications are sometimes called web-based software, on-demand software, or hosted software. Whatever the name, SaaS applications run on a SaaS providers servers. Instead of installing and maintaining software, you simply access it via the Internet, freeing yourself from complex software and hardware management. The provider manages access to the application, including security, availability, and performance. SaaS business applications are usually accessed by users using a thin client via a web browser.

Advantages of SaaS:

1. High adoption
2. Lower initial costs
3. Painless upgrades
4. Seamless integration.
SaaS facilitates remote application hosting and delivery, making this the key advantage of SaaS: painless application access. SaaS customers have no hardware or software to buy, install, maintain, or update. Access to applications is easy-you just need an Internet connection.
